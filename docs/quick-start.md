# AssetWise 同步功能快速入门

## 🚀 5分钟快速设置

### 第一步：启用同步
1. 打开 AssetWise 应用
2. 点击右上角 **设置** 图标
3. 选择 **同步设置**
4. 点击 **启用同步** 按钮

### 第二步：选择同步频率
- **推荐**：选择 **实时同步** 获得最佳体验
- **节能**：选择 **每15分钟** 平衡性能和电池

### 第三步：配置同步内容
默认同步所有内容，也可以自定义：
- ✅ 资产数据
- ✅ 分类标签  
- ✅ 用户设置
- ✅ 自定义字段

### 第四步：完成首次同步
- 点击 **立即同步** 按钮
- 等待首次同步完成（通常1-3分钟）
- 看到 ✅ **同步完成** 提示即可

## 📱 多设备使用

### 在新设备上设置
1. 下载并安装 AssetWise
2. 使用相同账户登录
3. 启用同步功能
4. 等待数据自动下载

### 数据会自动同步
- 添加新资产 → 自动同步到所有设备
- 修改资产信息 → 实时更新到云端
- 删除资产 → 同步删除操作

## ⚡ 常用操作

### 手动同步
- 下拉刷新主页面
- 或点击同步状态栏的刷新图标

### 查看同步状态
- 顶部状态栏显示同步进度
- 绿色 ✅ = 已同步
- 蓝色 🔄 = 同步中
- 红色 ❌ = 同步失败

### 解决同步冲突
如果出现冲突提示：
1. 点击冲突通知
2. 查看冲突详情
3. 选择保留哪个版本
4. 点击 **解决冲突**

## 🛠️ 故障排除

### 同步失败？
1. 检查网络连接
2. 确认账户登录状态
3. 重启应用重试
4. 查看错误恢复页面

### 数据丢失？
1. 访问 **版本历史** 页面
2. 选择最近的备份版本
3. 点击 **恢复** 按钮
