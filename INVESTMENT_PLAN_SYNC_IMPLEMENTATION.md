## 同步功能特性

### 1. 数据同步
- **双向同步**: 支持本地到云端和云端到本地的数据同步
- **冲突解决**: 实现了多种冲突解决策略（版本比较、时间戳检查）
- **增量同步**: 只同步变更的数据，提高效率
- **离线支持**: 离线时数据存储在本地，联网后自动同步

### 2. 同步状态管理
- **实时状态**: 显示网络连接状态、同步进度、待同步项目数量
- **权限检查**: 根据用户订阅级别控制同步功能访问
- **错误处理**: 完善的错误处理和用户提示

### 3. 数据版本控制
- **版本管理**: 每次数据修改都会创建新版本
- **冲突检测**: 通过校验和和时间戳检测数据冲突
- **历史记录**: 保留数据变更历史，支持版本回滚

## 技术实现

### 1. 数据流
```
用户操作 → 本地存储 → 同步队列 → Supabase云端
         ↑                              ↓
    UI更新 ← 状态管理 ← 同步服务 ← 云端数据
```

### 2. 同步策略
- **自动同步**: 数据变更后2秒自动触发同步
- **手动同步**: 用户可手动触发上传/下载
- **批量同步**: 支持批量处理多个数据项
- **重试机制**: 失败的同步操作会自动重试

### 3. 数据结构
投资计划数据结构包含：
- **基本信息**: 标题、描述、分类、优先级、风险等级
- **财务数据**: 目标金额、当前金额、预期收益率
- **时间规划**: 开始日期、结束日期
- **资产配置**: 资产名称、配置比例、当前价格、目标价格
- **执行计划**: 里程碑标题、目标日期、完成状态、详细描述
- **状态管理**: 计划状态（草稿、活跃、暂停、完成）
- **同步元数据**: 版本、校验和、设备ID、创建/更新时间

## 使用说明

### 1. 创建投资计划
1. 点击"新建计划"按钮
2. 填写基本信息（标题、描述、分类等）
3. 设置财务目标和时间规划
4. 配置资产组合和比例
5. 设定执行里程碑
6. 保存后自动同步到云端

### 2. 管理投资计划
1. 查看计划列表和统计概览
2. 使用搜索和筛选功能
3. 编辑计划详细信息
4. 更新计划状态和进度
5. 跟踪里程碑完成情况

### 3. 同步管理
1. 查看同步状态指示器
2. 手动触发"同步到云端"或"从云端拉取"
3. 监控同步进度和结果
4. 处理同步冲突（如有）

## 注意事项

1. **网络依赖**: 同步功能需要网络连接
2. **权限要求**: 需要专业版或旗舰版订阅
3. **数据安全**: 所有数据传输都经过加密
4. **存储限制**: 根据订阅级别有不同的存储限制
5. **配置验证**: 资产配置比例总和应为100%
6. **日期逻辑**: 结束日期必须晚于开始日期
