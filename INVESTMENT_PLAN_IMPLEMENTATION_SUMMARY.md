## 数据结构

### InvestmentPlan 接口
```typescript
interface InvestmentPlan {
  id: number
  title: string                    // 计划标题
  description: string              // 计划描述
  status: 'active' | 'paused' | 'completed' | 'draft'
  priority: 'high' | 'medium' | 'low'
  targetAmount: number             // 目标金额
  currentAmount: number            // 当前金额
  startDate: string               // 开始日期
  endDate: string                 // 结束日期
  expectedReturn: number          // 预期收益率
  riskLevel: 'low' | 'medium' | 'high'
  category: string                // 投资分类
  assets: AssetAllocation[]       // 资产配置
  milestones: Milestone[]         // 里程碑
  notes?: string                  // 备注
  createdAt: string              // 创建时间
  updatedAt: string              // 更新时间
  version?: number               // 版本号
  checksum?: string              // 校验和
  deviceId?: string              // 设备ID
}
```

### 资产配置
```typescript
interface AssetAllocation {
  name: string          // 资产名称
  allocation: number    // 配置比例
  currentPrice: number  // 当前价格
  targetPrice: number   // 目标价格
}
```

### 里程碑
```typescript
interface Milestone {
  title: string         // 里程碑标题
  targetDate: string    // 目标日期
  completed: boolean    // 完成状态
  description: string   // 详细描述
}
```

## 同步功能

### 1. 数据流
```
用户操作 → 本地存储 → 同步队列 → Supabase云端
         ↑                              ↓
    UI更新 ← 状态管理 ← 同步服务 ← 云端数据
```

### 2. 同步策略
- **自动同步**: 数据变更后2秒自动触发同步
- **手动同步**: 用户可手动触发上传/下载
- **冲突解决**: 版本比较和时间戳检查
- **离线支持**: 离线时数据存储在本地，联网后自动同步
- **权限控制**: 根据订阅级别控制同步功能访问

### 3. 状态管理
- **网络状态**: 实时显示在线/离线状态
- **同步进度**: 显示同步过程和结果
- **待同步计数**: 显示需要同步的项目数量
- **最后同步时间**: 记录和显示最后同步时间

## 用户体验

### 1. 创建投资计划流程
1. 点击"新建计划"按钮
2. **基本信息**: 填写标题、描述、分类、优先级、风险等级、预期收益率
3. **财务设置**: 设置目标金额、时间规划，查看投资概览
4. **资产配置**: 添加多个资产，设置配置比例（总和需为100%）
5. **执行计划**: 设定里程碑和时间节点
6. 保存后自动同步到云端

### 2. 管理投资计划
- **查看概览**: 统计卡片显示关键指标
- **搜索筛选**: 快速找到目标计划
- **状态管理**: 一键切换计划状态（草稿→活跃→暂停→完成）
- **进度跟踪**: 可视化进度条和里程碑状态
- **编辑更新**: 随时修改计划详情
- **数据同步**: 实时同步状态显示

### 3. 同步管理
- **状态指示**: WiFi图标显示网络状态
- **同步按钮**: 手动触发上传/下载
- **进度显示**: 旋转图标显示同步进行中
- **结果反馈**: Toast通知显示同步结果
